group by란 데이터를 기준이 되는 값으로 그룹을 묶어서 보여줘라..

한 칼럼에서 중복되는 데이터를 묶고 보여준다.

그룹으로 데이터를 묶었다는 것은 그룹안에 데이터들이 묶여있다는 것을 의미한다.

count => 한 그룹에 데이터가 얼마나 들어있느냐?
ex) 미국에 7명의 시민/ 중국에 10명의 시민
나라명으로 그룹바이를 하고 count를 하면 7, 10이 나오는 로직이다.

집계 함수
1. COUNT() : 행의 개수를 세어줌
2. AVG() : 행 안에 있는 값의 평균을 내어줌
3. MIN() : 행 안에 있는 값의 최솟값을 반환해줌
4. MAX() : 행 안에 있는 값의 최댓값을 반환해줌
5. SUM() : 행 안에 있는 값의 합을 내어줌
